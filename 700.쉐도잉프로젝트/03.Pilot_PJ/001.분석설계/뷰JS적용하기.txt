[ 뷰JS 적용하기 ]

1. 공통적용을 위한 페이지 구조화하기
    1) 데이터가 달라지는 부분 외엔 모두 같아지게함
    -> 남자,여자,스타일 3가지 페이지 변경시

        (1) 이미지 : 폴더만 다르고 이름은 같음
        -> 뷰JS에서 v-bind:src="'같은경로이름'+변수"
        -> 변수는 Vuex.Store를 사용함!!!
        -> 변수업데이트는 mutations 메서드로 작성함!
        -> 확장자일치 : 여기서는 png로 일치함!

        (2) 데이터 : 분류별 코드를 객체화 하여 데이터를 구조화한다!
        -> 3가지 분류별 객체를 만들고 데이터키를 속성명으로 생성하여
        구분되도록 만들고 사용하기 쉽게 한다!
        -> 뷰JS에서 각 요소의 특정위치에 표시되는 데이터값을
        매칭하여 v-html / v-text / {{}} 로 화면출력한다!
        -> 따라서 데이터 변경부분 외엔 요소의 구조가 모두 같게함!

    2) 반복되는 요소와 데이터는 변경사항에 맞게 객체데이터를 
    활용하여 반복구조를 생성한다.
    -> 뷰JS에서는 v-for 속성구문과 v-if 조건구문을 적절히 사용하여
    요구사항에 맞는 화면이 구성되게 한다.
    (이때 v-for를 쓴다면 반드시 v-bind:key를 사용할것!)
    ->> 키로 활용할 데이터는 v-for="(v,i) in 객체/배열"
    -> i는 배열일 경우 순번, 객체일 경우 속성명으로 유일키로 간주함!
